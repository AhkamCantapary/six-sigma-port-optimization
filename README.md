# 🚢 Reducing U.S. Port Congestion Using Six Sigma DMAIC

## 📌 Project Overview
This project applies the Six Sigma DMAIC framework to analyze inefficiencies in U.S. import logistics caused by port congestion and shifting trade routes.

## 🎯 Problem Statement
Major U.S. ports experienced severe congestion, leading to:
- Vessel wait times of up to 20–24 days  
- Increased logistics costs and delays  
- Reduced supply chain reliability  

## 🔍 Methodology (DMAIC)

### Define
Identified operational bottlenecks and inefficiencies in port logistics and import routing.

### Measure
Analyzed key metrics including:
- Vessel wait time  
- Port throughput  
- Container dwell time  

### Analyze
Identified root causes such as:
- Port congestion  
- Labor constraints  
- Throughput inefficiencies  

### Improve
Proposed solutions including:
- Process optimization  
- Improved scheduling  
- Reduced truck gate time (45 → 30 minutes)  

### Control
Recommended monitoring systems and control charts to sustain improvements.

## 📊 Key Results
- Identified major drivers of congestion  
- Proposed solutions to significantly reduce delays  
- Improved operational visibility and efficiency
